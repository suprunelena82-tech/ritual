# Світло Пам'яті — сайт ритуальної служби (демо)

Демонстраційний односторінковий сайт для пітчу клієнту. Усі тексти, телефон,
адреса та відгук — умовні, підлягають заміні на реальні перед публікацією.

## Структура

```
index.html
css/style.css
js/script.js
assets/        ← сюди класти реальні фото (логотип, світлини)
```

## Деплой через GitHub + Netlify

1. Створіть репозиторій на GitHub і залийте **всю** цю папку цілком,
   включно з `assets/` — якщо папку не запушити, зображення не завантажаться.
2. На netlify.com → **Add new site → Import an existing project**.
3. Авторизуйтесь через GitHub і оберіть цей репозиторій.
4. Build command можна лишити порожнім, Publish directory — `/` (корінь).
   Це статичний сайт, збірка не потрібна.
5. Після першого деплою Netlify видасть посилання виду `назва.netlify.app`.
   Кастомний домен підключається в Site settings → Domain management.

## Форма зворотного зв'язку

Зараз форма в `index.html` обробляється демонстраційним скриптом
(`js/script.js`) — просто показує підтвердження, нікуди не надсилає дані.

Щоб підключити реальний прийом заявок через Netlify Forms:
1. У тезі `<form ... id="contact-form">` додайте атрибут `data-netlify="true"`.
2. Додайте приховане поле `<input type="hidden" name="form-name" value="contact">`.
3. Netlify автоматично почне збирати заявки в розділі Forms — без бекенду.

## Заміна фото

Покладіть реальні фото в `assets/` (наприклад `assets/logo.svg`,
`assets/team.jpg`) і підключіть їх у `index.html` замість SVG-заглушок,
де це доречно.
